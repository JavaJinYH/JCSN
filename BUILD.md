
# 折柳建材管理系统 - 打包说明

## 打包前准备

### 1. 确保依赖已安装
```bash
npm install
```

### 2. 确保数据库存在
如果没有数据库，先运行一下项目，确保 `prisma/dev.db` 文件存在：
```bash
npm run dev
# 然后关闭，确保数据库已创建
```

## 打包命令

### Windows 打包
```bash
npm run electron:build:win
```

### 通用打包
```bash
npm run electron:build
```

## 打包输出

打包完成后，安装文件会在 `release` 目录下：
- `折柳建材管理系统-v1.0.0-setup.exe` - 安装程序
- `折柳建材管理系统-v1.0.0-portable.exe` - 便携版程序

## 安装说明

### 安装程序
1. 双击运行 `折柳建材管理系统-v1.0.0-setup.exe`
2. 选择安装路径（建议非系统盘）
3. 安装完成后，桌面会生成快捷方式
4. 双击运行即可

### 便携版
1. 直接运行 `折柳建材管理系统-v1.0.0-portable.exe`
2. 无需安装，解压即用

## 数据存储位置

### 安装版
- 应用程序：安装目录
- 数据库文件：`C:\Users\[用户名]\AppData\Roaming\折柳建材管理系统\database\dev.db`
- 照片文件：`安装目录\photos`

### 便携版
- 应用程序：同一目录
- 数据库文件：程序目录下的 `data` 文件夹
- 照片文件：程序目录下的 `photos` 文件夹

## 备份和恢复

### 备份数据
1. 打开系统设置 → 数据管理
2. 点击「备份数据」按钮
3. 保存备份文件到安全位置

### 恢复数据
1. 打开系统设置 → 数据管理
2. 点击「恢复数据」按钮
3. 选择之前保存的备份文件
4. 确认恢复（会覆盖现有数据）

### 手动备份
也可以直接复制数据库和照片文件：
- 数据库文件：`C:\Users\[用户名]\AppData\Roaming\折柳建材管理系统\database\dev.db`
- 照片文件：应用目录下的 `photos` 文件夹

## 常见问题

### 打包失败
- 确保已安装所有依赖
- 确保 `prisma/dev.db` 文件存在
- 确保有足够磁盘空间（需要 200MB+）

### 安装后无法运行
- 检查是否有杀毒软件拦截
- 尝试右键 → 以管理员身份运行
- 查看错误日志：`%APPDATA%\折柳建材管理系统\logs`

### 数据丢失
- 检查是否删除了应用目录
- 使用系统的备份恢复功能
- 检查数据库文件是否损坏

### 照片无法显示
- 检查 `photos` 文件夹是否在正确位置
- 检查照片文件是否完整
- 尝试重建照片索引（系统设置 → 数据管理 → 重建照片索引）

## 更新系统

1. 备份现有数据
2. 下载新版本安装包
3. 安装新版本（会覆盖旧版本）
4. 恢复数据（如果需要）

## 技术支持

如有问题，请查看系统日志或联系技术支持。
