## 日常使用流程

### 场景一：卖货（最常用）

```
1. 点击「销售」→ 点击「新增销售」
2. 选择客户/联系人
3. 添加商品（输入数量）
4. 确认金额无误
5. 选择付款方式（现金/赊账）
6. 点击「保存」
```

**详细步骤**：
1. **选择购货人**：点击下拉框，选择"水电工张师傅"等
2. **选择结账主体**：选择付款方，可以是个人或公司
3. **添加商品**：
   - 点击"添加商品"
   - 输入商品名称或选择
   - 输入销售数量
   - 单价自动显示（也可修改）
4. **选择付款方式**：
   - **现金**：当场收款，无需后续操作
   - **赊账**：挂到客户账户，后续催款
5. **保存**：点击保存按钮

### 场景二：进货（补充库存）

```
1. 点击「进货」→ 点击「新增进货」
2. 选择供应商
3. 添加商品（数量、单价）
4. 点击「保存」
```

**注意**：
- 进货后库存自动增加
- 如果商品不存在，先去「商品管理」添加

### 场景三：客户管理

```
1. 点击「客户」→ 点击「添加客户」
2. 填写客户信息
3. 保存
```

**客户类型**：
- **普通客户**：普通买家
- **水电工**：可以享受折扣，还能获得回扣
- **公司**：单位客户，通常赊账

## 功能模块说明

### 首页仪表盘

显示今日经营概况：
- 今日销售额
- 今日利润
- 库存预警数量
- 待收款金额

### 库存管理

**查看库存**
- 点击「库存」查看所有商品库存
- 使用筛选器按分类查看
- 使用搜索框搜索商品

**库存预警**
- 点击「库存预警」按钮
- 查看库存不足的商品
- 根据建议补货量进货

**盘点库存**
- 点击「库存盘点」
- 定期核对实际库存
- 发现差异可调整

### 商品管理

**添加商品**
1. 点击「商品」→ 「添加商品」
2. 填写信息：
   - 商品名称
   - 选择分类
   - 规格型号
   - 单位（根/米/个等）
   - 成本价（进货价）
   - 销售价
3. 设置库存预警线

**多规格商品**
- 一个商品可以有多个规格
- 例如：PPR热水管 - 20mm/25mm/32mm
- 每个规格可以设置不同价格

**价格波动商品**
- 某些商品价格经常变动（如电线）
- 开启"价格波动"标记
- 退货时按市场价计算

### 客户管理

**客户价值评分**
系统自动评估客户价值（★/△/○/×）：
- ★★★：优质客户
- △△：普通客户
- ○○：一般客户
- ××：需注意

**评分依据**：
- 年度累计销售额
- 平均利润率
- 回款速度
- 欠款逾期率

### 联系人管理

**联系人 vs 客户**
- **联系人**：个人（水电工、业主等）
- **客户**：可以是个人或公司
- 一个公司可以有多个联系人

**添加联系人**
1. 点击「联系人」→ 「添加联系人」
2. 填写姓名、手机号
3. 选择类型（customer/plumber/other）
4. 可添加多个手机号

### 结账主体管理

**什么是结账主体？**
- 付款方，可以是个人或公司
- 账目挂在主体下，不是挂在联系人下
- 便于统一管理和催款

**添加主体**
1. 点击「结账主体」→ 「添加主体」
2. 选择类型（个人/公司）
3. 关联联系人（如有）
4. 设置信用额度

### 项目管理

**项目用途**
- 跟踪大型工程的材料使用
- 例如："龙湖天街水电工程"

**项目状态**
- 进行中：正在施工
- 已完成：工程结束

### 进货管理

**进货退货**
- 如果发现问题货品，可以退货
- 常规商品按原进货价退货
- 价格波动商品按市场价退货

### 销售管理

**销售单四角色**
- **购货人**：去提货的人
- **付款人**：付钱的人
- **介绍人**：介绍业务的人（水电工）
- **提货人**：实际提货的人

**赊账销售**
- 选择赊账后，生成应收账款
- 后续在「挂账结算」中收款

**销售退货**
- 点击已有销售单 → 「退货」
- 选择退货商品和数量
- 库存自动返还

### 回扣管理

**什么是回扣？**
- 给水电工的介绍费
- 按销售单或季度结算

**登记回扣**
1. 点击「回扣」→ 「添加回扣」
2. 选择关联销售单
3. 填写供应商名称、回扣金额
4. 选择回扣类型（现金/转账）

### 配送管理

**添加配送单**
1. 点击「配送」→ 「添加配送」
2. 关联销售单
3. 填写收件人、地址、距离、重量
4. 配送费自动计算

**配送费计算**
- 基础费 + 距离费 + 重量费
- 可在「配置配送费」中设置费率

### 挂账结算

**应收账款**
- 查看所有客户欠款
- 按账龄分级（30天内/30-90天/90-180天/180天以上）
- 颜色标识：绿色/黄色/橙色/红色

**还款操作**
1. 点击「还款」
2. 输入还款金额
3. 确认还款

**部分还款**
- 如果客户只还一部分，选择部分还款
- 剩余欠款继续挂在账户

### 催账记录

**记录催款过程**
1. 点击「催账记录」→ 「添加催账」
2. 选择关联客户和欠款
3. 选择催账方式（电话/微信/上门）
4. 选择催账结果（已承诺付款/继续催/无法联系）

### 对账单

**生成对账单**
1. 选择客户
2. 选择年度
3. 点击「生成对账单」

**对账单内容**
- 客户基本信息
- 欠款总金额
- 已还金额
- 剩余欠款
- 账龄分布
- 未结清挂账明细

**批量操作**
- 勾选多个客户
- 批量导出或打印对账单

### 报表统计

**销售日报**
- 查看某一天的销售额、订单数、利润

**商品销售排行**
- 按销量/销售额/毛利排序
- 分析哪些商品卖得好

**利润统计**
- 查看成本、收入、毛利
- 分析经营状况

**导出报表**
- 点击「导出Excel」
- 生成表格文件，方便进一步分析

### 照片管理

**查看销售单照片**
- 点击「照片」
- 选择类型（销售单/进货单）
- 查看关联的照片

**批量导出**
- 选择多张照片
- 导出到本地文件夹

### 系统设置

**店铺信息**
- 设置店铺名称、联系方式
- 打印时显示店铺信息

**数据备份**
- 定期备份数据库
- 防止数据丢失

## 常见问题

### Q: 如何修改商品价格？
**A**: 商品管理 → 找到商品 → 点击编辑 → 修改价格 → 保存

### Q: 客户欠了多少钱怎么查？
**A**: 挂账结算 → 查看应收账款列表

### Q: 销售单可以修改吗？
**A**: 已保存的销售单不能直接修改，但可以退货后重新开单

### Q: 如何给客户打折？
**A**: 在销售开单时，可以手动修改单价

### Q: 水电工的回扣怎么算？
**A**: 在回扣管理中登记，关联到具体的销售单

### Q: 数据存在哪里？
**A**: 数据保存在 `%APPDATA%\jiancai-manager\prisma.db`

### Q: 如何备份数据？
**A**: 复制整个 `%APPDATA%\jiancai-manager\` 文件夹

## 注意事项

1. **赊账要谨慎**：赊账前确认客户信用
2. **定期对账**：每月核对一次应收账款
3. **及时催款**：欠款超过90天要积极催收
4. **数据备份**：重要数据定期备份
5. **价格更新**：市场行情变化时及时调整售价
