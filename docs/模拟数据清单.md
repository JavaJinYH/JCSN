## 十三、快速录入提示

### 分类批量添加
在商品管理 → 9模块 → 添加分类，可以一次性输入多个分类：
```
电线电缆
水管管材
卫浴洁具
五金配件
工具配件
```

### 同一供应商多个商品
在添加进货时，可以一个订单里添加多个商品。

### 同一客户多个销售
可以分别开多个销售单，比如：
- 一个散客今天买了A商品
- 明天又买了B商品 → 开另一个销售单
